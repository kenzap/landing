## Setup Instructions

1. **Initialize project**:
```bash
npm create astro@latest kenzap-factory -- --template minimal --yes
cd kenzap-factory
```

2. **Install dependencies**:
```bash
npm install
npm install -D tailwindcss @astrojs/tailwind
npm install @tailwindcss/postcss
npx astro add tailwind
```

3. **Create the file structure** and copy the files I'll provide below

4. **Run development server**:
```bash
npm run dev
```

5. **Build for production**:
```bash
npm run build
```

6. **Deploy** to Netlify, Vercel, or any static hosting

## Quick Setup Guide

1. **Create the project**:
```bash
npm create astro@latest kenzap-factory -- --template minimal --yes
cd kenzap-factory
```

2. **Install Tailwind**:
```bash
npx astro add tailwind
```

3. **Create all directories**:
```bash
mkdir -p src/{components,i18n,layouts,pages/de,pages/fr,styles}
```

4. **Copy all files** from the artifact above into their respective locations

5. **Update your LinkedIn URL** in `src/components/Contact.astro` (line 78)

6. **Run dev server**:
```bash
npm run dev
```

7. **Build for production**:
```bash
npm run build
```

The site will be available at:
- English: `http://localhost:4321/`
- German: `http://localhost:4321/de`
- French: `http://localhost:4321/fr`

---

## Netlify Deployment

1. Push your code to GitHub
2. Connect to Netlify
3. Build settings:
   - Build command: `npm run build`
   - Publish directory: `dist`
4. The contact form will work automatically with Netlify Forms (no additional config needed!)
